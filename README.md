# Python Calculator

A simple command-line calculator developed using Python. The program takes two numbers and an arithmetic operator as input and performs the selected operation.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Technologies Used

- Python 3

## Concepts Demonstrated

- User Input
- Variables
- Conditional Statements (if-elif-else)
- Arithmetic Operations

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/codeakash-cyber/basic-python-calculator.git
```

2. Navigate to the project folder:

```bash
cd basic-python-calculator
```

3. Run the program:

```bash
python calculator.py
```
